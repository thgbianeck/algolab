# AlgoLab — Laboratório de Algoritmos em Python

## Descrição

Projeto prático desenvolvido ao longo do curso
"Algoritmos do Básico ao Avançado com Python".
O objetivo é implementar, do zero, os principais algoritmos
da ciência da computação com código claro, comentado e testado.

## Tecnologias

- Python 3.13.x
- VS Code
- pytest
- Windows 11

## Princípio de Independência

Cada aula é autocontida. O código de uma aula
não depende do código de aulas anteriores.
Você pode estudar qualquer aula de forma isolada.

## Como Executar os Testes

1. Ative o ambiente virtual:
   .venv\Scripts\activate

2. Navegue até a pasta da aula:
   cd modulo_01_fundamentos\aula_01_o_que_e_algoritmo

3. Execute os testes:
   pytest testes/ -v

## Módulos

- Módulo 1: Fundamentos de Algoritmos e Python (Aulas 1-7)
- Módulo 2: Busca e Ordenação (Aulas 8-14)
- Módulo 3: Estruturas de Dados (Aulas 15-21)
- Módulo 4: Grafos, Paradigmas e Complexidade (Aulas 22-28)
